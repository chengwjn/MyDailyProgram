### 运算符重载

c++允许将运算符重载扩展到用户定义的类型，例如，允许使用+将两个对象相加。编译器将根据操作数的数目和类型决定哪种加法定义。
重载运算符可使代码看起来更自然。例如将两个数组相加使一种常见的运算。<br>
通常，需要使用下面这样的for循环来实现：
```C++
for(int i=0;i<20;i++)
	evening[i]=sam[i]+janet[i];//add element by element
```
