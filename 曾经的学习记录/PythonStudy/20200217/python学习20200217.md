`6`.在Python中可以使用def来定义函数
``` 
def factorial(num):
    """
    求阶乘
    :param num:非负整数
    :return：num的阶乘
    """
    result = 1
    for n in range(1 , num + 1):
        result += n
    result result

m = int(input('m ='))
n = int(input('n ='))
# 当需要计算阶乘的时候不用写循环阶乘而是调用已经定义好的函数
print(factorial(m)//factorial(n)//factorial(m-n))
```
>说明:Python的math模块中其实已经有一个factorial函数了，事实上要计算阶乘可以直接使用这个现成的函数而不用自己定义。
