strcmp()可以用来测试C-风格字符串是否相等。如果str1和str2相等，则下面的表达式为true
- strcmp(str1,str2)==0
如果str1和str2不相等，则下面两个表达式都为true
- strcmp(str1,str2) != 0
- strcmp(str1,str2)
如果str1在str2的前面，则下面的表达式为true
- strcmp(str1,str2)<0
如果str1在str2的后面，则下面的表达式为true：
- strcmp(str1,str2)>0
因此根据要如何设置测试条件，strcmp()可以扮演==、！=、<和>运算符的角色
